# Git/GitHub チュートリアル

研究室に新しく配属された学生向けの Git/GitHub チュートリアル．

## 実施方法

先生役の先輩は最低でも二人いるとよい（司会進行役と巡回役）．
また，生徒役の B4 は二人一組のペアを作って協力するとよい．
Git を使うメリットを理解するためにも，多少プログラミングをやってからの方がいいかもしれない．
簡単なプログラミング演習をした後に Git を学び，それ以降のプログラミング演習は各自 Git を使って行うとよい練習になると思う．

## 各 Step の概要

以下に Step 1–5 の概要を記す．
司会進行役の先輩はこの順番に講習会を進めればよい．

### Step 1

- 詳細は `Step1.md` を参照
- Git/GitHub の概念を知ろう
- 環境構築
  - Git のインストール
  - Git の設定
- GitHub アカウントの作成とローカル環境との連携
  - GitHub のアカウント作成
  - GitHub CLI のインストール
